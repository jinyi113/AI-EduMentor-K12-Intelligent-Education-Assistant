# 🏆 AI-EduMentor：K12 全场景智能教育陪伴平台
## 广东省大学生计算机设计大赛 · 高职组 参赛作品

---

## 📌 项目概述
AI-EduMentor 是一款面向 K12 教育场景的全链路智能教育助手，基于 OpenClaw 低代码智能体平台开发，同时通过 GitHub 完成项目托管与版本管理。
产品深度覆盖**学生、教师、家长**三类核心用户，一站式解决「学习盲目、批改繁重、家校脱节」三大教育痛点，实现「诊断-辅导-练习-复盘-协同」的完整学习闭环。

---

## 🎯 核心痛点与解决方案
| 角色 | 核心痛点 | AI-EduMentor 解决方案 |
|------|----------|------------------------|
| 学生 | 刷题盲目、知识漏洞难定位、课后答疑不及时 | 智能作业批改+知识点诊断+个性化学习路径+7×24小时AI答疑 |
| 教师 | 作业批改耗时、学情分析效率低、分层教学难落地 | 批量自动批改+班级学情可视化报告+分层教学建议 |
| 家长 | 无法精准掌握孩子学习状态、辅导力不从心 | 自动学情周报+家庭教育指导+学习进度实时同步 |

---

## 🚀 核心功能模块
### 1. 智能作业批改系统
- 支持客观题、主观题、作文全类型自动批改
- 标注对应知识点、错误原因、改进建议
- 支持图片/PDF/手写体多格式输入识别

### 2. 知识点诊断与可视化
- 自动分析错题，定位知识薄弱点
- 生成知识点掌握度雷达图、学情报告
- 班级/个人多维度学情分析

### 3. 个性化学习路径生成
- 基于学情动态生成专属学习计划
- 推送定制化习题、微课、知识点精讲
- 难度循序渐进，适配不同学习水平

### 4. 7×24 小时 AI 答疑辅导
- 多学科知识点通俗化讲解
- 举一反三同类题拓展训练
- 鼓励式沟通，保护学习积极性

### 5. 家校协同服务
- 自动生成学生学习周报
- 家庭教育指导建议
- 家校学习进度实时同步

## 🛠️ 部署与使用指南
### 1. OpenClaw 智能体部署
1.  登录 OpenClaw 平台，新建智能体
2.  导入 `/openclaw/prompt.txt` 完整提示词
3.  上传 K12 学科知识库（教材大纲、真题、知识点）
4.  启用文档解析、公式识别、图表生成等插件
5.  发布智能体，生成分享链接
